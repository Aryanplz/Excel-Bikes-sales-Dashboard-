# 🚴 Bike Sales Data Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing **customer bike purchasing behavior** using Microsoft Excel. The dataset was initially messy and unstructured, requiring extensive **data cleaning and transformation** before analysis.

After preparing the dataset, **Pivot Tables and interactive visualizations** were used to build a dynamic dashboard that provides insights into how demographic and lifestyle factors influence bike purchases.

The goal of this project is to demonstrate **data cleaning, exploratory data analysis, and dashboard creation skills using Excel**.

---

# 🎯 Business Problem

A company selling bicycles wants to understand:

* Which **customer segments are more likely to purchase bikes**
* How **income levels affect purchasing behavior**
* Whether **commuting distance influences bike purchases**
* Which **age groups are the most active buyers**

Understanding these patterns helps businesses improve **target marketing and sales strategies**.

---

# 📂 Dataset

The dataset contains customer demographic and lifestyle information such as:

* Customer ID
* Gender
* Age
* Marital Status
* Education
* Region
* Income
* Commute Distance
* Bike Purchase Status

The raw dataset required **cleaning and transformation** before analysis.

---

# 🧹 Data Cleaning Process

Several preprocessing steps were performed:

* Removed inconsistencies in categorical values
* Standardized text formatting
* Created **Age Brackets** for segmentation
* Structured the dataset for **Pivot Table analysis**
* Verified data consistency for accurate insights

---

# 📊 Data Analysis

Using **Pivot Tables**, key metrics were calculated to identify trends in bike purchases.

### Key Analysis Performed

**1️⃣ Average Income per Purchase**

* Compared average income of customers who purchased bikes vs those who did not.
* Analyzed differences between **male and female customers**.

**2️⃣ Customer Age Brackets**

* Segmented customers into:

  * Adult
  * Middle Age
  * Old
* Identified which age groups purchase bikes the most.

**3️⃣ Customer Commute Distance**

* Examined how commute distance impacts bike purchasing decisions.

---

# 📈 Dashboard Features

The final dashboard provides **interactive insights** through:

* **Income Analysis Chart**
* **Age Bracket Purchase Trends**
* **Commute Distance Analysis**
* **Interactive Filters (Slicers)**

Users can filter data by:

* Marital Status
* Region
* Education Level

This allows dynamic exploration of customer behavior.

# 📊 Key Insights

Some insights derived from the analysis include:

* Customers with **higher income tend to purchase bikes more frequently**.
* **Middle-aged customers** represent the largest group of bike buyers.
* People with **short commute distances** show higher interest in bike purchases.
* Demographic factors such as **education and marital status** also influence buying behavior.
